## 🎨 Design Highlights

### Color Usage
- **Background**: Cream (#F5F2EA) - Primary
- **Accent**: Blue (#1E4FD8) - Headlines, primary CTAs
- **Highlight**: Yellow (#F2B305) - Labels, emphasis
- **Dark**: #1A1A1A - Dark sections, text
- **Borders**: 2px solid black - Bauhaus style

### Typography
- **Headings**: Archivo Bold, large sizes (48px-72px)
- **Subheadings**: Archivo Bold, medium sizes (24px-36px)
- **Body Text**: Archivo Regular, 14-16px
- **Code/Terminal**: Monospace font, syntax highlighted
- **Line Height**: 1.4-1.6 for comfort

### Visual Elements
- **Borders**: 2px solid, no radius (Bauhaus)
- **Shadows**: Hard shadows (no blur), 4-8px offset
- **Animations**: 0.6s ease-out transitions
- **Icons**: Lucide React, 16-32px sizes
- **Spacing**: Consistent 16px baseline grid
